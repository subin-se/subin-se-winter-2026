GitHub Pages 构思
构思维度	引导问题与记录建议	参考思路（契合你的目标）
1. 站点定位	一个持续成长的自主学习者的数字花园”，主要观众是未来的自己，其次是潜在的学习伙伴。
2. 核心板块	
		• 🏠 首页：个人简介、寒假总目标、本周聚焦。
		• 📓 学习周报：每周复盘报告的集合（直接链接至GitHub Issues）。
		• 📚 双考语料库：精选的词汇、句子、分析范例。
		• 🎨 作品集：PPT截图、Vlog视频、画作的展示区。
		• 🚀 项目：你的GitHub仓库动态。
3. 内容来源
		• “学习周报”板块 → 自动链接到你的 Issues 中带 weekly-review 标签的Issue。
		• “双考语料库” → 定期从OneNote中精选部分内容，整理成Markdown文档放入站点。
		• “作品集” → 将图片、视频上传至仓库的 /assets 目录，在站点中引用。
4. 风格与命名	
		名称：Subin's Learning Hub 或 知行录。
